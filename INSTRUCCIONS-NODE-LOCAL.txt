INSTRUCTIONS:

INSTALL NODE

1. Open the Terminal
Press the Windows key, type PowerShell, and select Run as administrator.

2. Find and install Node.js
Type the following command to install the LTS version (the most stable and recommended):


winget install OpenJS.NodeJS.LTS


3. Accept the terms
During the process, the console will ask if you accept the license agreements. Press the Y key and then Enter.

4. Restart the console and verify
It's important to close the PowerShell window and reopen it so that the new system paths are recognized. Once it's open again, check that everything is correct:

PowerShell

node -v
npm -v
If you get back the version numbers (for example, v20.x.x), you have Node.js installed and ready to go.

1. Create the invisible script
Open Notepad and paste this code:

Set WshShell = CreateObject(“WScript.Shell”)
WshShell.Run “cmd.exe /c cd /d ”“[insert here the complete URL of the location of the project folder]‘’ && npm start”, 0
Set WshShell = Nothing


2. Save the file
Go to File > Save As.

In the “Save as type” dropdown, select “All files (*.*)”.

Name it: `start_node.vbs` (it's very important that it ends with .vbs).

Save it in a safe place (for example, in the same folder as your project).

3. Set up automatic startup
Press the Windows key + R.

Type shell:startup and press Enter. A folder will open.

Copy and paste your start_node.vbs file inside this folder.

4. Verification
Restart your PC or simply double-click the .vbs file you created.

Since it's invisible, you won't see any window. To confirm it's working, open your browser and go to your application's address (e.g., localhost:3000).
